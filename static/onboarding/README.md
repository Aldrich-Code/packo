# Packo 首次启动引导图

请把引导图导出为 PNG，放在本目录：`static/onboarding/`。

推荐尺寸：`1600 x 1000 px`

显示比例：`16:10`

安全区：核心文字和箭头尽量放在中间 `1400 x 820 px` 内，边缘留白，避免小窗口裁切。

## 文件清单

1. `welcome.png`
   - 内容：Packo 首页两个卡片 + 拖拽入口示意。
   - 目的：告诉用户 Packo 可以压缩、解压、预览、编辑。

2. `file-tree-drag-zones.png`
   - 内容：解压窗口文件树局部，重点标出三种拖拽热区。
   - 标注建议：
     - `拖图标：拖出到 Finder`
     - `拖名称：调整包内结构`
     - `拖选其他字段：批量选择`
   - 目的：说明图标、名称、类型/大小/日期列拖拽含义不同。

3. `context-menu.png`
   - 内容：压缩包文件树右键菜单示意。
   - 重点：重命名、删除、新建文件夹、打开方式。

4. `default-open.png`
   - 内容：将 Packo 设为默认打开方式的示意。
   - 重点：双击压缩包直接用 Packo 打开。

## 视觉建议

- 用真实界面局部截图或高保真重绘，不要用太抽象的插画。
- 高亮色使用 Packo 的浅黄色。
- 每张图里的文字尽量短，不要超过 3 个标注点。
- 箭头、框选和高亮圈要明显，但不要遮挡文件名。
