# Packo 功能路线图

更新时间：2026-06-14

## 状态说明

- `[ ]` 未开始
- `[~]` 进行中
- `[x]` 已交付

## 阶段 1：基础持久化

- [x] 最近文件持久化，下次启动仍保留
- [x] 设置持久化：默认格式、默认保存/解压位置、是否排除 `.DS_Store`
- [x] 解压文件树列宽记忆
- [x] 列排序记忆
- [x] 基础快捷键：打开、压缩、解压、搜索、设置

## 阶段 2：解压体验补齐

- [x] 批量解压多个压缩包
- [x] 解压冲突策略：覆盖、跳过、重命名、全部应用
- [x] 分卷压缩包识别：`.001`、`.z01`、`.part1.rar`
- [x] 解压安全检查：路径穿越、隐藏文件、可执行文件提示
- [x] 解压完成通知和 Finder 定位

## 阶段 3：压缩任务增强

- [x] 批量压缩任务队列
- [x] 7Z 分卷压缩
- [x] 7Z 非加密压缩参数补齐
- [x] 高级参数：字典大小、固实压缩、线程数、压缩方法
- [x] 压缩完成后自动测试压缩包完整性
- [x] 支持排除 `.DS_Store`
- [x] 支持排除 `__MACOSX`、扩展属性、资源分叉等 macOS 元数据

## 阶段 4：压缩包浏览能力

- [x] 压缩包内搜索、排序、过滤
- [x] 压缩包属性面板：压缩率、CRC、压缩方法、加密状态、分卷信息
- [x] 测试压缩包完整性
- [x] 密码需求检测，提前标记加密文件
- [x] 文件预览缓存和清理策略

## 阶段 5：压缩包编辑

- [x] 在压缩包内删除文件
- [x] 在压缩包内重命名文件
- [x] 追加文件/文件夹
- [x] 替换已有文件
- [x] 编辑后重新保存或另存为

## 阶段 6：任务中心

- [ ] 所有压缩/解压进入任务队列
- [ ] 支持并发数设置
- [ ] 暂停、继续、取消、重试
- [ ] 失败详情和日志窗口
- [ ] 速度、剩余时间、失败数量、当前文件展示
- [ ] 取消后清理残留文件

## 阶段 7：macOS 集成

- [x] 文件关联：双击压缩包用 Packo 打开
- [x] Finder 右键菜单：压缩、解压到当前目录、解压到同名文件夹
- [x] Dock 拖拽压缩/解压
- [x] 菜单栏完整中文菜单和快捷键
- [ ] 系统通知
- [ ] Quick Look 预览插件

## 阶段 8：高级功能和 polish

- [ ] 深色模式
- [ ] 收藏常用目录
- [ ] 默认解压目录规则
- [ ] 压缩预设：最快、均衡、最小体积、加密
- [ ] 校验值计算：MD5、SHA256、CRC32
- [ ] 自动解压后删除原压缩包
- [ ] 自动压缩后校验
- [ ] 命令行入口

## 交付记录

- 2026-06-14：建立路线图文档。
- 2026-06-14：已交付 `.DS_Store` 排除选项，勾选后文件树隐藏并且压缩包不包含 `.DS_Store`。
- 2026-06-14：已交付本地偏好持久化，最近文件、默认格式、默认保存/解压位置、`.DS_Store` 选项和解压文件树列宽会在下次启动保留。
- 2026-06-14：已交付基础快捷键：打开压缩包、选择压缩项目、执行解压、打开最近文件搜索、打开设置和关闭弹窗。
- 2026-06-14：已交付解压文件树列排序和排序记忆，名称、类型、大小、修改时间支持点击表头切换升降序。
- 2026-06-14：已交付批量解压多个压缩包，选择或拖入多个压缩包后只需选择一次目标文件夹，Packo 会为每个压缩包创建同名子文件夹并显示总进度。
- 2026-06-14：已交付解压冲突策略，解压页可选择覆盖已有文件、跳过已有文件或自动重命名，策略会随偏好保存并应用到单包和批量解压。
- 2026-06-14：已交付分卷压缩包识别，支持识别 `.001`、`.z01`、`.part1.rar` 等分卷入口，并会提示用户选择 ZIP 主文件或第一卷。
- 2026-06-14：已交付解压安全检查，压缩包列表会提示隐藏项目、可执行项目和不安全路径，不安全路径会在解压/预览/拖出前被后端阻止。
- 2026-06-14：已交付解压/压缩完成通知和 Finder 定位，任务完成后会发送系统通知，完成弹窗可直接在 Finder 中显示输出位置。
- 2026-06-14：已交付阶段 3 压缩增强：支持顶层项目批量压缩队列、7Z 分卷、7Z 非加密高级参数、压缩后完整性测试，以及排除 `__MACOSX`、`._*`、扩展属性和资源分叉等 macOS 元数据。
- 2026-06-14：已交付阶段 4 压缩包浏览增强：包内搜索/过滤、属性面板、ZIP 条目 CRC/压缩方法/加密标记、完整性测试和预览缓存清理。
- 2026-06-14：已交付阶段 5 压缩包编辑：支持删除、重命名、追加文件/文件夹、替换已有文件，以及原地保存或另存为。
- 2026-06-14：已交付阶段 7 的 macOS 集成主体：文件关联、Dock 拖拽打开/压缩、Finder 服务菜单，以及完整中文菜单栏和快捷键；Quick Look 和系统通知按要求暂不处理。
